App Listing -- FabMo Apps in Progress and Example Apps

FabMo Apps will eventually be available from sources appropriate for particular Fabmo-powered tools and equipment. For Handibot, that will be the App and Project Gallery on the Handibot website. For the moment, apps are still works in progress and we are experimenting with many types. The list below links you to apps currently under active development. Try them out on your FabMo system and learn from them.

The first app listed below is the definitive reference example with documentation explaining the structure of apps ... and functioning in a illustrative fashion.

Most of the apps include a current install version in the form of an .fma file. These are a zipped version of all the necessary files with a file extension of .fma.

  • THE DEFINITIVE INTRO and EXAMPLE APP... with documentation [Ryan Sturmer]: example-app

  • Example app creating TEXT cut-outs for things like signs or house numbers [Jonathan Ward]: txtmaker-app

  • Sb3-Style Interface simulating legacy ShopBot software ... early in development [Ted Hall]: sb4-app

  • Touch and Go! A quick "touch-and-go" interface for FabMo systems that controls the XY table on your machine [Ryan Sturmer]: touchandgo-app

  • App for cutting spoilboards for ShopBot tools[Ryan Sturmer]: spoilboarder-app

  • Demo app for creating example CNC cuts [Ryan Sturmer]: partmaker-app

  • App for cutting programmatically generated terrain surfaces [Ryan Sturmer]: terrainman-app
